# 5.0 - Navegação Contextual

## Objetivo
- Facilitar o fluxo de trabalho do analista, levando-o diretamente do histórico de uploads para a ação de conciliação.

## Escopo / Entregáveis
- Alteração no comportamento do botão "Processado" em `public/js/upload.js`.

## Passos e subtarefas
- 5.1 Localizar evento de clique no status "Processado" da tabela de histórico.
- 5.2 Implementar troca programática de aba para "Similaridade".
- 5.3 Passar o `uploadId` para o componente de similaridade para disparar o carregamento automático.

## Dependências
- 4.0 - UI de Conciliação (Frontend).

## Paralelizável?
- Sim, após a estrutura básica da 4.0 estar definida.

## Critérios de aceite
- Ao clicar em "Processado" no Histórico, o usuário deve "aterrizar" na aba de Similaridade já vendo os itens daquele upload específico.

## Testes
- 5.4 Validar o redirecionamento entre abas sem recarregar a página (SPA behavior).

## Notas
- Utilizar a API do Bootstrap: `new bootstrap.Tab(element).show()`.
